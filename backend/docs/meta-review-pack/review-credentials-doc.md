# Meta App Review Credentials Doc

Required runtime values:

- `META_APP_ID`
- `META_APP_SECRET`
- `META_OAUTH_STATE_SECRET`
- `INSTAGRAM_VERIFY_TOKEN`
- `WHATSAPP_VERIFY_TOKEN`
- `BACKEND_URL`
- `FRONTEND_URL`

Reviewer notes:

- Use Meta test business assets in reviewer mode seed path.
- Do not use production secrets in screencasts.
- Rotate credentials after review window closes.
